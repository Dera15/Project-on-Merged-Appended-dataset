UK Crime Data Merged & Analysis 

A data analysis project that consolidates multi-year UK crime records into a single, clean dataset for easier exploration and reporting.

Project Highlights

- Merged crime data spanning multiple financial years into one structured dataset.
- Standardized columns and formats to ensure consistency across records.
- Prepared data for downstream analytics, visualization, and machine learning tasks.
- Demonstrates practical skills in Excel, Power query, Append, data cleaning, and ETL workflows.

Dataset

The dataset contains information such as:

- Financial Year & Quarter
- Police Force
- Offence Description
- Offence Group & Subgroup
- Offence Code
- Number of Offences

Purpose

This project showcases my ability to transform raw, fragmented data into an analysis-ready format while following a structured and reproducible workflow.
